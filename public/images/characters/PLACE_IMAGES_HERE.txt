📁 CHARACTER IMAGES GO HERE!

The game is currently using placeholder images. To add real character images:

1. Download character images from Genshin Wiki or official sources
2. Create folders for each character using their ID (e.g., "albedo", "venti", "diluc")
3. Place 4 images in each character folder:
   - portrait.png (200x300px) - Main character image
   - icon.png (64x64px) - Small face icon  
   - side-icon.png (64x64px) - Side view icon
   - wish.png (300x400px) - Banner/wish image

Example structure:
characters/
├── albedo/
│   ├── portrait.png
│   ├── icon.png
│   ├── side-icon.png
│   └── wish.png
├── venti/
│   ├── portrait.png
│   ├── icon.png
│   ├── side-icon.png
│   └── wish.png

The game will automatically use your images when they're placed correctly!
